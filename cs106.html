Hello CS 106 Students!

Now that you've completed CS 106a, one direction you can explore with your newfound Python knowledge is data science and machine learning!

Next quarter, STATS 112 / DATASCI 112 will be offered. This is a new course designed to introduce the field of data science to freshmen and sophomores. Unlike other data science and machine learning courses on campus, the only prerequisite is CS 106a; there are no math or statistics prerequisites. That is because this class will be focused on the practical, rather than theoretical, aspects of data science. This website contains more information about the course: http://dlsun.github.io/stats112.

This course is great for anyone who is considering the new major in Data Science or just interested in learning what Data Science is about (and maybe picking up some useful skills). If you do end up majoring in Data Science, this class will count towards the elective requirements for the major. Even if you do not, this class fulfills the WAY-AQR requirement.

If you have any questions, please don't hesitate to e-mail Professor Sun at dlsun@stanford.edu.
