Course Description

This course is a practical overview of the field of data science. We will manipulate and analyze data of different shapes and sizes (e.g., tabular, textual, hierarchical, geospatial). We will discuss common patterns and pitfalls of data analysis. We will understand the essence of machine learning and train machine learning models on real data. This course culminates in a project where you apply the ideas from this class to a data science problem of your choosing.

Who Should Take This Class?

This course is designed for undergraduates early in their college careers.

  • This is a hands-on class that will have you analyzing real data. To do this, you need to know how to code (preferably in Python). So CS 106a is a prerequisite. You do not need to be an experienced programmer; if you know how to write a for loop and how to use a dict in Python, you'll be fine.
  • You need to be comfortable with basic math. We will be aggregating, transforming, and comparing numbers every day in this class. No calculus, no linear algebra. We will focus on the intuition behind data science methods, not the math.
If you have a lot of experience with data science and machine learning already, this is probably not the class for you. You will likely learn some useful skills, but you might find the pace of the class slow.

Class Structure

There will be some kind of class every weekday:

  • On Mondays, Wednesdays, and Fridays, we will meet as an entire class for lecture. In lecture, the professor will introduce data science concepts, do some live coding, and assign some exercises for you to try at home.
  • On Tuesdays and Thursdays, you will meet with a TA in small sections. In section, you will present and discuss solutions to the exercises I posed the day before.

Attendance is mandatory and counts towards your participation grade (see below).
