Course Description

This course is a practical overview of the field of data science. We will manipulate and analyze data of different shapes and sizes (e.g., tabular, textual, hierarchical, geospatial). We will discuss common patterns and pitfalls of data analysis. We will introduce the key ideas of machine learning and learn how to train machine learning models on real data. This course culminates in a project where you apply the ideas from this class to a data science problem of your choosing.

Prerequisites

This course is designed to be accessible to undergraduates early in their college careers. As such, there are no math prerequisites. That said, data science is a highly quantitative field. You will be aggregating, transforming, and comparing numbers all time, but nothing will involve "advanced" math. But you need to be comfortable with high school algebra.

However, this course is hands-on. You will be doing data science, so you need to know how to code. So there is one programming prerequisite: CS 106a. You can probably get by if you have prior experience in Python (or a scientific programming language, like Matlab or R).

Class Structure

There will be some kind of class every weekday:

  • On Mondays, Wednesdays, and Fridays, we will meet as an entire class for lecture. In lecture, the professor will introduce data science concepts, do some live coding, and assign some exercises for you to try at home.
  • On Tuesdays and Thursdays, you will meet with a TA in small sections. In section, you will present and discuss solutions to the exercises I posed the day before.

Attendance is mandatory and counts towards your participation grade (see below).
